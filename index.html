Project

Aurbit is an Interplanetary map based strategy game using non-fungible Avatar ERC721 tokens, a fungible ERC777 resource AUR token, a Governance contract, and Planet based game contracts. Each planet has its own map and players acquire land pixels. Holding a pixel mines AUR tokens for every new block and pixels can be attacked and taken by other players.

Game

Avatars

Aurbit uses a Non Fungible token contract on Ethereum to keep unique DNA for Human and Alien Races. Each Race has unique attribues and user selected qualities. The Avatars are called AURA and are required to enter gameplay. They hold the tokens that are mined.

Game Mechanics

Games take place on planets with unique geographies based on pixelized maps. Earth is the first and default game world where coordintates of the map can be owned, or attacked. Holding pixels will mint to the Avatar contract AUR tokens, which can be allocated to a coordintate to bolster defence against attack.

Smart Contracts

Aurbit is based on Ethereum and uses ERC 721 Non Fungible Collectible tokens for avatars. The game resource is an ERC 777 token that is minted and burned during gameplay. Each planet in the game is a unique game contract with many possibilities and variations.

Avatars

Avatars are unique using ERC721 non-fungible tokens. The DNA of the avatar includes the following parameters:

[
haircolor[0],
haircolor[1],
haircolor[2],
eyecolor[0],
eyecolor[1],
eyecolor[2],
skincolor[0],
skincolor[1],
skincolor[2],
hairtype,
eyetype,
skintype,
mouth,
gender,
race
]

Based on these values, we build a Unique DNA integer for the Avatar that looks like this:

[76,98,65, 228, 239, 117, 68, 239, 5, 4, 239, 153, 5, 2, 9, 3, 85]

The AvatarAUR contract stores the DNA as a byte array which is used to render and keep the unique state of the avatar. The token ID of the avatar in the ERC 721 contract is used as the user ID in the Planet.sol (game) contract.

Game Mechanics
Buy

Unclaimed map pixels can be bought or ETH, however, if the Pixel is owned, the cost is 3x. The previous owner receives the fee.

Von Neumann

If you buy a pixel diagonal to a held pixel, you will own the two immediate Von Neumann neighbors. This is similar to how the Go board game.

Aerial Attack

A pixel can be attacked whereby a randomized algorithm includes the DNA of the avatars, the tokens allocated to the pixel, and a home turf advantage. An attack burns AUR tokens for both players.
